# 分词器

可以使用以下接口查看各个分词器的分词效果。

```shell
GET _analyze
{
  "analyzer": "分词器名称",
  "text": "待分词的字符串"
}
```

## Standard

ES的默认分词器，根据单词进行切分，并把切分后的单词转为小写，停词不会被过滤。

## Simple

会把字符串中所有非字母的字符去掉，并按单词进行切分，停词不会被过滤。

## Whitespcae

根据空格进行切分。

## Stop

在Simple分词器的基础上加入停词过滤功能。

## Keyword

不对结果进行分词。

## Pattern

根据正则表达式进行分词。
